# internal/web

Встроенный одностраничный дашборд, доступный по адресу `/`.

HTML, JS и CSS встроены в `page.go` через `go:embed` (или строковую константу). UI подключается к `/api/v1/stream` через `EventSource` и перерисовывается при каждом SSE-сообщении без перезагрузки страницы.

Отображает:
- Метаданные кластера и лидерство.
- Кластерные группы и группы управления с состоянием desired/actual/health.
- Состояние ролей на каждом узле.
- Статус регистрации и сессий подключённых агентов.

Не реализовано: управление кластером через UI, редактор конфигурации, просмотр истории/diff.
