# internal/logging

Создание структурированных логгеров и тегирование компонентов.

Формирует `zap.Logger` из конфигурации (уровень, plain/JSON формат, вывод в консоль и/или файл). `WithComponent` добавляет поле `component` к экземпляру логгера, чтобы строки логов из разных пакетов были различимы без добавления дополнительного поля в каждом месте вызова.
